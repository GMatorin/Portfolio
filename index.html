Обо мне

Я начинающий Frontend-разработчик, недавно закончивший своё высшее образование в Германии. Во время учёбы я работал программистом на различных проектах, на одном из которых мне довелось столкнуться с миром веб разработки. С этого момента я понял, что создание современных веб приложений и их визуальной составляющей это моё призвание. Если Вам нужен молодой специалист с огромным интересом к веб разработке, опытом работы в команде над различными проектами и постоянно изучающий новые технологии, то Вы можете связаться со мной по телефону или e-mail на Ваш выбор.

Мои проекты

Здесь вы можете взглянуть на некоторые из моих личных проектов

ImageFinder

Приложение для нахождения бесплатных фотографий.

  • Реализация при помощи React JS
  • Context API для обновления стейтов и передачи данных
  • Загрузка фото при помощи Axios
  • Создание стейтов в функциональных компонентах с React Hooks
  • Вёрстка с компонентами от Material UI и Bootstrap для React
Проект Github

Local Coffee

Лэндинг для кофейни.

  • Адаптивная реализация сайта под разные размеры экранов
  • Стили прописаны при помощи SASS
  • Для компиляции CSS файла был использован node-sass
  • Самостоятельное планирование UI/UX сайта и создание макета в Adobe XD
  • Интеграция Google карты
Проект Github

CitiesFight

Приложение для сравнения городов США по численности населения.

  • Использование только нативного JavaScript
  • Применение функций высшего порядка из ES6
  • Fetch API для загрузки данных
  • Bootstrap вёрстка
  • Локализация багов при помощи Google developer tools
